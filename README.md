Program ini berfungsi untuk menghitung biaya pengiriman paket berdasarkan berat, jarak, dan ukuran paket.
Cara kerjanya yaitu pengguna memasukkan data berat, jarak, panjang, lebar, dan tinggi paket. 
Program kemudian menghitung volume, menentukan tarif per kilogram berdasarkan jarak, lalu menghitung biaya kirim.
